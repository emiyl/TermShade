# TermShade

TermShade is a small macOS menu bar app that switches Apple Terminal themes based on light/dark mode.

Pick one theme for light mode and one for dark mode, and TermShade keeps Terminal in sync.

## What it does

- Shows in the menu bar
- Lets you choose light and dark Terminal themes
- Auto-applies when macOS appearance changes
- Tries to update all open Terminal tabs

## Requirements

- macOS
- Terminal.app
- Xcode

## Build and Run

1. Open TermShade.xcodeproj in Xcode.
2. Select the TermShade scheme.
3. Build and run.
4. Click the menu bar icon and choose your light/dark themes.

## Usage

1. Choose a light mode theme.
2. Choose a dark mode theme.
3. Switch macOS appearance to test.

If Terminal is open, TermShade also tries to apply the theme to open tabs right away.
